ExcelBundle
===========
##H2 Getting started
######H6 Install the bundle using composer
Add following lines to composer.json 
    `{
            "type": "git",
            "url": "git@github.com:Tactics/ExcelBundle"
    }`
    
    "tactics/excel-bundle": "1.0.*"

######H6 Getting excel file from excel template.
